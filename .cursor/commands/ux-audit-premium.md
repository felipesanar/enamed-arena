---

Use the ux-ui-architect subagent.

Perform a deep premium UX/UI audit of the current screen, route, feature, or relevant flow.

This is not a shallow design review.
This is a high-standard product experience audit.

Analyze the experience through these lenses:

## 1. User Journey
- What is the user trying to do?
- What is the likely mental model?
- What happens before this screen?
- What happens after this screen?
- Where does friction exist in the journey?
- Where can the user feel uncertainty, hesitation, or cognitive overload?

## 2. UX Quality
- clarity of next step
- discoverability
- confidence and trust
- friction points
- cognitive load
- error prevention
- error recovery
- feedback quality
- task completion flow
- time-to-value
- empty/loading/error state quality

## 3. UI Quality
- first impression
- visual hierarchy
- CTA prominence
- scanability
- spacing rhythm
- alignment precision
- typography hierarchy
- density balance
- content grouping
- layout composition
- visual consistency
- premium feel
- trust cues

## 4. Interaction Quality
Evaluate all relevant states and interactions:
- default
- hover
- focus
- active
- selected
- disabled
- loading
- empty
- success
- warning
- error
- skeleton
- motion and transitions

Assess whether interactions feel:
- clear
- responsive
- intentional
- polished
- satisfying

## 5. Responsiveness and Accessibility
- desktop quality
- mobile quality
- touch comfort
- information priority on smaller screens
- keyboard/focus behavior
- contrast
- readability
- semantics
- form clarity

## 6. Design System and Product Coherence
- consistency with the rest of the product
- token usage
- component reuse
- repeated inconsistencies
- opportunities to improve patterns systemically

## Required Output
Structure the audit like this:

### Premium UX/UI Audit

#### What is working
List the strongest aspects briefly.

#### Main problems
List the most important UX/UI problems in priority order.

#### Highest-impact improvements
List the best improvements that would most strongly elevate the experience.

#### Premium opportunities
Call out opportunities that would make the experience feel significantly more world-class, refined, and memorable.

#### Recommended implementation plan
Break into:
- quick wins
- medium improvements
- deeper structural improvements

Be highly critical, precise, and practical.
Do not give generic advice.
Do not stop at visual commentary.
Think like an elite product design partner.
