---

Use the ux-ui-architect subagent.

Redesign and implement the current screen, component set, or relevant flow at a world-class premium quality bar.

I do not want cosmetic cleanup.
I want a real product experience upgrade.

## Mission
Elevate this experience to the highest realistic level by improving:
- user journey clarity
- visual hierarchy
- layout composition
- spacing rhythm
- CTA quality
- interaction states
- responsiveness
- accessibility
- design system coherence
- perceived polish
- premium feel

## Required approach

### 1. Audit first
Before editing, inspect:
- current layout
- component structure
- nearby flow
- user goal
- friction points
- weak hierarchy
- weak interaction patterns
- missing states
- responsiveness issues
- accessibility gaps

Briefly explain:
- what is wrong
- what should improve
- what the redesign strategy is

### 2. Implement the redesign
Then implement directly in code.

Prioritize:
- clearer information hierarchy
- better spacing and alignment
- more refined density
- stronger CTA emphasis
- better grouping of information
- more premium component treatment
- better affordances
- better transitions and micro-interactions
- better state design
- better mobile behavior
- better confidence and task completion feel

### 3. Evaluate every state
Ensure the redesign considers all relevant states:
- default
- hover
- focus
- active
- selected
- disabled
- loading
- empty
- success
- warning
- error
- skeleton

### 4. Respect the system
- reuse existing components and patterns where possible
- strengthen the design system instead of creating unnecessary inconsistency
- preserve maintainability and performance
- preserve accessibility semantics

### 5. Polish pass
After implementation, do an additional polish pass.
Refine:
- spacing rhythm
- visual hierarchy
- motion quality
- state quality
- microcopy clarity if needed
- visual calmness
- premium coherence

### 6. Final review
Before finishing, critically assess:
- does this now feel premium?
- is the journey clearer?
- are interactions more satisfying?
- is hierarchy stronger?
- does the experience feel intentional and product-grade?
- is mobile meaningfully improved?
- are states complete?

If not, refine further before stopping.

## Response format
Use this structure:

### Redesign strategy
Brief explanation of the redesign intent and priorities.

### Implemented improvements
Summarize the most important changes made.

### Final polish notes
Summarize the refinements that elevated the result from good to premium.

Be decisive, demanding, and high-standard.
Aim for an exceptional product experience, not just a nicer screen.
