# Skills do projeto

Skills são carregadas automaticamente pelo Cursor. Cada subpasta deve conter um `SKILL.md` com frontmatter `name` e `description`.

Para importar skills a partir de um boilerplate `.agent`:

```powershell
.\.cursor\scripts\transpose-agent-to-cursor.ps1 -AgentPath "C:\caminho\para\.agent"
```

Veja `.cursor/TRANSPOSE-AGENT-TO-CURSOR.md` para o mapeamento completo rules/skills/workflows → Cursor.
